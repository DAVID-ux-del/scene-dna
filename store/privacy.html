Privacy Policy — SceneDNA

Last updated: 2026-07-13

本扩展没有开发者运营的中转服务器。开发者不收集、不存储你的图片、提示词、API Key 或浏览记录。

我们收集什么

不收集任何个人信息。开发者无法看到你的图片、提示词、API Key 或浏览记录。

数据如何流动

当你右键图片并选择「用 SceneDNA 解析图片」时：扩展在本地读取该图片字节，连同你的 API Key 发送到你自己配置的 API 地址（默认 AiHubMix）；返回的提示词显示在页面浮层中。请求只发往你配置的第三方 API 服务，开发者不经手任何内容。

API Key 存储

你的 API Key 通过 chrome.storage.local 仅保存在本机，不同步云端，不发往除你配置 API 地址以外的任何地方，可随时清除。

第三方

不集成任何分析、广告或追踪 SDK。

权限

扩展申请的权限仅用于核心功能：添加右键菜单、读取被右键的图片、注入结果浮层、向你配置的 API 发请求。

联系

如有疑问，请通过 Chrome 应用商店的开发者联系方式与我们联系。
