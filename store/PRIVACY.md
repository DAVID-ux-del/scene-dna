# Privacy Policy — SceneDNA

_Last updated: 2026-07-13_

SceneDNA（"the extension"）尊重并保护你的隐私。本扩展没有开发者运营的中转服务器，开发者**不收集、不存储**你的图片、提示词、API Key 或浏览记录。

## 我们收集什么

**不收集任何个人信息。** 开发者无法看到你的图片、提示词、API Key 或浏览记录。

## 数据如何流动

当你在某张图片上右键并选择「转换为 GPT Image 提示词」时：

1. 扩展在你的浏览器本地读取该图片的字节；
2. 连同你的 API Key，发送到**你自己在设置中配置的 API 地址**（默认 AiHubMix，`https://aihubmix.com/v1`）；
3. 返回的提示词显示在当前页面浮层中。

以上请求**只发往你自己配置的那个 API 服务**。开发者不参与、不经手、不留存任何内容。图片和 API Key 会被传输给你配置的第三方 API 服务，并适用该服务的隐私政策。

## API Key 的存储

你的 API Key 通过 Chrome 的 `chrome.storage.local` 保存，**仅存在于你本机**，不会同步到云端，不会发送到除你所配置 API 地址以外的任何地方。你可随时在设置中清除。

## 第三方

本扩展不集成任何分析、广告或追踪 SDK。

## 权限

扩展申请的浏览器权限仅用于其核心功能（添加右键菜单、读取被右键的图片、注入结果浮层、向你配置的 API 发请求），不用于其它目的。

## 变更

本政策如有更新，将更新本页顶部的日期。

## 联系

如有疑问，请通过 Chrome 应用商店的开发者联系方式与我们联系。
